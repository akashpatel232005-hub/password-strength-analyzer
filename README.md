#  Password Strength Analyzer

A modern web-based Password Strength Analyzer that helps users evaluate the security of their passwords in real time.

## Features

- Real-time password strength analysis
- Strength indicator with colored progress bars
- Password requirements checklist
- Show/Hide password functionality
- Strong password suggestions
- One-click password copy feature
- Responsive and user-friendly UI

## Password Rules Checked

The analyzer checks for:

- Minimum 8 characters
- Uppercase letters (A-Z)
- Lowercase letters (a-z)
- Numbers (0-9)
- Special characters (!@#$%^&*)
- 12+ character bonus for stronger passwords
